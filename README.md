 Sign Language Detection System

## Overview

This project detects Indian Sign Language (ISL) hand gestures using a webcam and converts them into text using Machine Learning.

## Features

* Real-time hand detection using MediaPipe
* Hand landmark extraction
* Machine Learning based sign classification
* Live webcam prediction
* Supports alphabet recognition (A-Z)
* Built using Python

## Tech Stack

* Python
* OpenCV
* MediaPipe
* Scikit-Learn
* NumPy
* Pandas
